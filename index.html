Notch

What started as a personal score-keeping system eventually morphed into a full-fledged iOS app that keeps record of who is best (or worst) at games. Once more data was being entered into the database, with some of the more complex stats, we eventually needed something to speed up queries. Turning to SQL views for the first time, it was a lot of trial and error, but they significantly eased database load times. I did dabble with ReactJS for the front-end, but I mainly worked on building the JSON API with all of the SQL views, and all the nerdy stats no one cares about.

  • Rails
  • SQL
  • JSON API
  • ReactJS
